The gpbbase GPB Module
The gpbbase GpbModule defines the scalar types that are allowed by the Google Protocol Buffer (GPB) Domain Specific Lanaguage (DSL). All .gpb modules have a dependency on this module by default.

GpbScalarType definitions cannot be defined in any other module! Attempting to do so will result in a parsing error.

Scalar types
bool Represents boolean values.
bytes May contain any arbitrary sequence of bytes.
double Represents double values.
fixed32 Represents fixed length integers.
fixed64 Represents fixed length long integers.
float Represents float values.
int32 Represents integer values using variable length encoding.
int64 Represents long integer values using variable length encoding.
sfixed32 Represents fixed lnegth, signed integers. Always four bytes.
sfixed64 Represents fixed lnegth, signed long integers. Always eight bytes.
sint32 Represents signed integers using variable-length encoding.
sint64 Represents signed long integers using variable-length encoding.
string A string that always contains UTF-8 encoded or 7-bit ASCII text.
uint32 Represents unsigned integers using variable-length encoding.
uint64 Represents unsigned long integers using variable-length encoding.
